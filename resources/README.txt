Выгрузка ключей VK для перевода на 10.11.2018
Выгружены секции со страницы https://vk.com/translationIntoLanguage
Секция geoDB при открытии с этой страницы не содержит ключей, поэтому секция не выгружена
Файлы содержат секции, отсортированные по алфавиту.
Поле section name содержит имя секции, section key — ключ, russian — текст секции, description — комментарий (если есть).
Поля разделены символом "табуляция".
Файлы можно открыть в любом текстовом редакторе или Excel (данные --> из файла, указать разделитель = табуляция и кодировку = utf-8)
Поле is_translated содержит флаг, указывающий на то, переведена ли секция для текущего языка (выгрузка делалалсь для нанайского).
В некоторых случаях комментарий содержит отсылку к скриншоту, но скриншоты не выгружались.
Символы \n в текстах секции заменены на <br>
